# Skills-Repo

A personal collection of reusable Codex skills for daily work.

个人 Codex 技能仓库，用于沉淀日常工作中可复用的技能与流程。

## Skills

### fix-image-text-typos

**English**

Corrects Chinese typos, OCR mistakes, wrong characters, and awkward Chinese text embedded in raster images such as PNG, JPG, screenshots, infographics, poster exports, and diagrams.

The workflow focuses on preserving the original visual design: transcribe and verify the text, build a correction list, edit only the affected text regions, save a non-destructive corrected image copy, and visually verify the final result for readable text and layout fit.

**中文**

用于修复 PNG、JPG、截图、信息图、海报导出图、流程图等栅格图片中的中文错别字、OCR 误识别字符、错误用字和不自然中文表达。

该技能强调尽量保留原始视觉设计：先转写并校对图片文字，整理修正清单，只编辑受影响的文字区域，另存为非破坏性的修正版图片，并对最终图片进行视觉复查，确保文字清晰、排版不重叠、整体布局仍然协调。
